1. 새로 배운 내용 : 
웹 페이지를 데이터베이스와 연동해 코딩할 때 변수 앞에 $를 붙여야 한다는 사실을
실습을 통해 확실하게 인지할 수 있었고, php 코드 자체를 처음 접해봤기 때문에
이렇게 DB 제어 부분 (PHP) 와 페이지를 표현하는 부분 (HTML) 을 분리하는 것이
굉장히 새롭게 느껴졌다. 

2. 문제가 발생하고 고민한 내용 :
우선 서버가 localhost 로 코딩할 경우 mysql과 연동되지 않아서 슬랙을 참고해 localhost:3370으로 코딩하여
연동에 성공할 수 있었다. 또, 직접 과제를 할 때 select 구문이 적용되지 않아 난관에 봉착했었는데 
이는 임의로 설정한 변수 명을 착각해서 일어난 문제였으며 코드를 하나하나 여러번 읽어본 후에 
오류를 수정할 수 있었다.
